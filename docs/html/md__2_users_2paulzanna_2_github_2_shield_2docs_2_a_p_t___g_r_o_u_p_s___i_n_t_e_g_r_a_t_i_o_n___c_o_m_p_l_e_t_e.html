MITRE ATT&CK Groups Integration - COMPLETE

Overview

The MITRE ATT&CK Groups integration has been successfully implemented as the next most important FREE data source for Shield's early warning cyber attack system. This integration provides APT threat actor intelligence to enhance existing threat assessment capabilities by tracking known APT groups and their country attributions.

Implementation Details

1. APT Groups Data Collection

  • Source: MITRE ATT&CK Framework Groups dataset
  • Collector: backend/geopolitics/aptGroupCollector.py
  • Coverage: 6 major APT groups with country attributions
  • Update Frequency: On-demand via --apt-only flag

2. Database Schema

The apt_groups table stores comprehensive APT group information:

  • Primary Fields: group_id, name, aliases, description
  • Attribution: country_code, country_name (ISO 3-letter codes)
  • Capabilities: techniques_count, software_count, sophistication_level
  • Activity: first_seen, last_seen, is_active
  • Context: primary_motivation, sectors_targeted, regions_targeted

3. Threat Scoring Integration

APT groups data is integrated into the cyber threat scoring algorithm:

  • History Assessment (25% weight): Now uses actual APT attribution data
  • Sophistication Scoring: Advanced = 15pts, High = 10pts, Medium = 5pts
  • Activity Bonus: Up to 15 points for active groups
  • Country Mapping: 2-letter to 3-letter country code conversion

Usage Examples

Collect APT Groups Only

python3 run_data_collectors.py --apt-only

Full Pipeline with APT Integration

python3 run_data_collectors.py --countries 10

Query APT Groups Programmatically

from backend.geopolitics.aptGroupCollector import APTGroupCollector
from backend.database import SessionLocal
db = SessionLocal()
apt_collector = APTGroupCollector(db)
# Get all APT groups for Russia
russia_apts = apt_collector.get_country_apt_groups('RUS')
for apt in russia_apts:
print(f"{apt['name']}: {apt['sophistication_level']}")
