Detailed Description

TypeScript API Utils

Utility functions for working with TypeScript's API. Successor to the wonderful tsutils. 🛠️️

Usage

npm i ts-api-utils
import * as tsutils from "ts-api-utils";
tsutils.forEachToken(/* ... */);
