Detailed Description

Unity Test Framework

This directory contains the Unity test framework for C unit testing.

Files

  • unity.h - Main header file with test macros and assertions
  • unity.c - Implementation of Unity test framework
  • unity_internals.h - Internal definitions and structures

Usage

Include unity.h in your test files and link with unity.c.

#include "unity.h"
void setUp(void) {
// Set up test fixtures
}
void tearDown(void) {
// Clean up after tests
}
void test_example(void) {
TEST_ASSERT_EQUAL(1, 1);
}
int main(void) {
UNITY_BEGIN();
RUN_TEST(test_example);
return UNITY_END();
}
