Detailed Description

MPU-6050 IMU driver with health checks and calibration.

Driver for MPU-6050 6-axis IMU providing gyroscope and accelerometer data with health monitoring and calibration support per Scout firmware requirements.

Perform level calibration of MPU-6050.

Performs stationary level calibration to determine gyro and accelerometer biases. Device must be placed on a level surface and remain stationary during calibration.

Parameters
devMPU-6050 device structure
callbackOptional callback for progress updates (can be NULL)
user_dataUser data passed to callback
Returns
ESP_OK on success, ESP_ERR_INVALID_STATE if motion detected
