Detailed Description

CLI save command implementation.

Provides configuration persistence functionality using NVS storage with atomic operations and proper error handling per CLI contracts.

Features:

  • Atomic commit of all configuration parameters
  • NVS storage with checksum validation
  • Proper error handling for storage failures
  • CLI contracts compliance (SAVED output)
