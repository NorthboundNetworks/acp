Detailed Description

Shield

Overview

Shield is a full-stack web application for geopolitical risk and stability analysis. It collects, integrates, and analyzes real-time financial, news, and cybersecurity data for 250+ countries, providing composite stability and threat scores. The system is designed for researchers, analysts, and organizations needing up-to-date insights into global stability, security, and cyber risk.

Key features:

  • Real-time data collection from financial and news APIs
  • Automated stability and cyber threat scoring for all countries
  • React frontend with interactive map and data visualizations
  • FastAPI backend with SQLite database and SQLAlchemy ORM
  • Extensible architecture for new data sources and analytics

Data Sources

  • World Bank API: Financial indicators (GDP growth, inflation, unemployment, government debt, etc.)
  • News APIs: Real news articles from multiple sources, with sentiment and relevance scoring
  • ITU Global Cybersecurity Index: Cybersecurity capability and posture
  • MITRE ATT&CK Groups: APT threat actor intelligence with country attributions
  • Country Metadata: Demographics, region, peace index (from public datasets)

Scores

Stability Score

  • Description: Composite score (0-100) reflecting a country's overall stability
  • Algorithm: Combines political, economic, social, and security factors:
    • Political Stability: Government effectiveness, rule of law
    • Economic Stability: GDP growth, inflation, unemployment
    • Social Cohesion: Social unrest, inequality
    • Security Index: Crime rates, conflict indicators
  • Calculation: Weighted sum using real data and news sentiment (see backend/geopolitics/geopolAnalytics.py)

Cyber Threat Score

  • Description: Composite score (0-100) estimating a country's potential cyber threat
  • Algorithm: Based on the framework in spec/threat_sources.md:
    • Capability (30%): Technical infrastructure, GCI scores
    • Intent (25%): Geopolitical posture, peace index
    • History (25%): Past cyber conflicts, APT activity
    • Exposure (20%): Economic vulnerability, digital dependency
  • Calculation: Weighted sum, higher = greater threat potential

Application Design

A full-stack web application with a React frontend and FastAPI backend using SQLite database.

Getting Started

Backend Setup

  1. Navigate to the backend directory:
    cd backend
  2. Create a virtual environment:
    python -m venv venv
    source venv/bin/activate # On Windows: venv\Scripts\activate
  3. Install dependencies:
    pip install -r requirements.txt
  4. Run the FastAPI server:
    uvicorn main:app --reload --port 8000

The API will be available at http://localhost:8000 with documentation at http://localhost:8000/docs.

Frontend Setup

  1. Navigate to the frontend directory:
    cd frontend
  2. Install dependencies:
    npm install
  3. Run the development server:
    npm run dev

The React app will be available at http://localhost:5173.

API Endpoints

  • GET /api/items - Retrieve all items
  • POST /api/items - Create a new item
  • GET /api/countries - List all countries and status
  • GET /api/countries/{country_code} - Get status for a specific country
  • GET /api/cyber_threat_top10 - Top 10 countries by cyber threat score
  • GET /api/cyber_threat_scores - All countries' cyber threat scores

Development

  • Both servers support hot-reloading for development.
  • The frontend is configured to proxy API requests to the backend server.
  • Database schema and data scripts are in docs/ and project root.

Features

  • React frontend with Vite build tool
  • FastAPI backend with automatic OpenAPI documentation
  • SQLite database with SQLAlchemy ORM
  • CORS enabled for cross-origin requests
  • Automatic database table creation
  • Real-time data collection and scoring
  • Extensible for new data sources and analytics
