Detailed Description

CLI SCP configuration command implementation.

Provides Secure Control Protocol (SCP) configuration management through CLI. Supports session management, security configuration, transport settings, and protocol statistics per CLI contracts.

Features:

  • Session listing and management
  • Security key configuration
  • Transport interface settings
  • Protocol statistics and diagnostics
  • Rate limiting configuration
