Cyber Threat Assessment Using Public Data

This document outlines publicly available data sources and analytical techniques to assess a nation's potential cyber threat capability and intent. It expands beyond financial and news feeds to incorporate technical, institutional, geopolitical, and behavioural indicators.

1. Technical Infrastructure & Indicators

Key Data

  • IP Address Ownership
    • ASNs, CIDR blocks, geolocation.
    • BGP announcements, peering relationships.
    • Sources:
      • RIPE NCC ✅, APNIC ✅, Hurricane Electric BGP Toolkit ✅
  • Exposed Infrastructure
    • Open ports, default creds, vulnerable services.
    • SSL/TLS cert fingerprints and misconfigs.
    • Tools: Shodan ⚠️ (advanced), Censys ⚠️ (advanced), ZoomEye ✅
  • Cybersecurity Research Output
    • Publications, patents, cyber tool development.
    • Sources: Semantic Scholar ✅, arXiv ✅, Google Scholar ✅
  • APT Infrastructure Mapping
    • Indicators of Compromise (IoCs)
    • Sinkhole, honeypot, and threat intel feeds
    • Sources: Abuse.ch ✅, MalwareBazaar ✅, ThreatFox ✅

2. Human Capital & Education

Key Metrics

  • Number of CS/IT/Engineering graduates
  • Government investment in cyber education
  • Participation in global hacking competitions

Indicators

  • CTF Participation & Ranking: CTFTime ✅
  • MOOC/Coding Platform Data: GitHub commits ✅, participation in HackTheBox ✅ and LeetCode ✅
  • STEM Index: Use UNESCO Institute for Statistics ✅, OECD Education Data ✅

3. Legal and Institutional Frameworks

Key Indicators

  • National cyber offence capability declarations
  • Data protection and surveillance law rigour
  • National CERT or military cyber command

Resources

  • ITU Global Cybersecurity Index ✅
  • National Cyber Strategy Repository (Atlantic Council) ✅

4. Geopolitical Posture & Conflict History

Factors

  • Past cyber conflict involvement (e.g. APT attribution)
  • State doctrine on information warfare
  • Military white papers referencing cyber operations

Tools & Sources

  • Council on Foreign Relations Cyber Tracker ✅
  • MITRE ATT&CK Groups ✅
  • EU Disinfo Lab ✅
  • NATO StratCom ✅
  • ACLED (Armed Conflict Location & Event Data) ✅

5. Economic & Industry Signals

Indicators

  • Size of domestic tech sector (esp. cybersecurity or exploit dev)
  • Internet penetration & digital economy dependence
  • Grey/shadow market activity indicators

Data Sources

  • World Bank ICT Indicators ✅
  • Akamai State of the Internet Report ⚠️ (full datasets)
  • Cisco Umbrella Data ⚠️ (full telemetry)

6. Social and Ideological Signals

Indicators

  • Rhetoric from state media or hacker groups
  • Forum activity: underground markets, nationalist hacking forums
  • Surveillance and censorship indices

Tools and Feeds

  • Freedom on the Net (Freedom House) ✅
  • DarkOwl ⚠️, Intel471 ⚠️, Telegram OSINT tools ✅

7. Open Source Intelligence (OSINT)

Signals

  • GitHub/Malware dev contributions
  • Data breaches (leaks, pastes)
  • Code similarities in malware samples

Platforms

  • GitHub ✅, GitLab ✅
  • HaveIBeenPwned ✅ (basic), ⚠️ (API)
  • VirusTotal ⚠️ (full API), Hybrid Analysis ⚠️ (sandbox automation)

8. Linguistic & Malware Artefacts

Detection Techniques

  • Compile times and time zone analysis
  • Language settings in binaries
  • TTP (Tactics, Techniques, Procedures) fingerprinting

Tools

  • YARA Rules ✅
  • MITRE ATT&CK Navigator ✅
  • Malpedia ✅ (academic access)

Composite Threat Scoring Framework

Categories

  • Capability: Infrastructure, STEM talent, malware tooling
  • Intent: Doctrine, state rhetoric, sanctions
  • History: Prior APT activity, conflicts
  • Exposure: Global interconnectivity, economic vulnerability

Suggested Algorithm

threat_score = (0.3 * capability_index +
0.25 * intent_index +
0.25 * history_index +
0.2 * exposure_index)

You can normalise each index on a 0–100 scale using z-scores or min-max scaling. Optionally, use a weighted AHP (Analytic Hierarchy Process) or PCA (Principal Component Analysis) to reduce dimensions for multivariate analysis.

Conclusion

A nation’s cyber threat potential is multi-dimensional. By systematically analysing open-source and technical data, a clearer risk picture can emerge. The approach outlined above enables proactive threat intelligence and strategic cyber defence planning.
