[0.6.4] - 2025-08-14

Added

  • Advanced Task Management System: Complete task lifecycle management with CRUD operations, real-time status updates, and visual progress tracking.
  • Sophisticated React Web UI: Professional React+TypeScript interface using shadcn/ui components with four-tab layout (Overview, Workflows, Deliverables, Activity & Audit).
  • Multi-Step Task Creation Wizard: KickoffWizard with 5-step configuration process (Scope, Literature, Experiment, Materials, Review) including localStorage persistence and Zod validation.
  • Resource Integration: Materials management with picklist integration, equipment scheduling with 14-day reservation forecasts, and DoE (Design of Experiments) integration.
  • Type-Safe API Client: Hybrid MCP+native database access pattern with Zod schemas for runtime validation and comprehensive error handling.
  • Real-time Data Synchronization: React Query with 5-second polling, automatic cache invalidation, and optimistic updates.
  • Protocol Management: Playbook preview with markdown rendering, BOM integration, and real-time cost tracking.

Enhanced Beyond Original Scope

  • Visual Status Management: Color-coded status chips with context-aware action controls (Start/Stop/Complete/Delete).
  • Materials Cost Tracking: BOM pricing with currency formatting and inventory management.
  • Equipment Timeline Visualization: Date-based reservation display with resource capability tracking.
  • Task Metadata System: Structured metadata for literature runs, experiment designs, and materials builds.

Changed

  • Task management evolved from minimal read-only UI to enterprise-grade workflow management system.
  • Database schema extended with comprehensive task lifecycle tracking and resource integration.
  • API endpoints enhanced with real-time capabilities and type-safe validation.

Notes

  • v0.6.4 represents production-ready task management functionality significantly exceeding original scope.
  • All backend services verified healthy with complete task lifecycle testing completed.
  • Frontend build system and API endpoints fully operational with end-to-end workflow verification.
