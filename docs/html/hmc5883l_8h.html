Detailed Description

HMC5883L 3-axis magnetometer driver with calibration and health checks.

Driver for HMC5883L 3-axis magnetometer providing magnetic field measurements with health monitoring, sphere calibration, and heading computation.

Perform sphere calibration of HMC5883L.

Performs 3D sphere fitting calibration to determine hard and soft iron corrections. Device should be rotated in all orientations during calibration period.

Parameters
devHMC5883L device structure
callbackOptional callback for progress updates (can be NULL)
user_dataUser data passed to callback
Returns
ESP_OK on success, ESP_ERR_TIMEOUT if insufficient samples, error code otherwise
