Detailed Description

Integration Tests for SCP Protocol Implementation.

Tests end-to-end command processing for Secure Control Protocol (SCP) v1.1, validating the complete flow from message reception through command execution and response generation.

Test Coverage:

  • Basic ARM/DISARM command processing
  • Mode change operations
  • Response generation (ACK/ERR)
  • Safety manager integration
  • Role-based authorization
  • Protocol timing requirements (<100ms)
  • Error handling scenarios
