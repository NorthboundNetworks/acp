Detailed Description

Scout ISR Drone System

Overview

Scout is a military-grade quadcopter drone platform developed for armed forces and first responders. Designed as a rapidly deployable, vehicle-launched Intelligence, Surveillance, and Reconnaissance (ISR) solution, Scout provides tactical overwatch and pre-arrival reconnaissance capabilities in austere or contested environments.

Key Features

  • Vehicle-Launchable:
    Designed for launch from moving vehicles (up to 80 km/h), Scout can be deployed from compact launch tubes (< 100 mm diameter), enabling rapid response during on-the-move operations.
  • **ISR-Capable:**
    Equipped with onboard sensors, gimballed cameras, and live telemetry systems, Scout provides real-time intelligence for situational awareness and decision support.
  • Autonomous Operation:
    Features pre-programmed flight paths, waypoint navigation, and automated return-to-launch functionality, minimizing operator workload.
  • Rugged & Reliable:
    Built to MIL-STD-810G standards, Scout withstands harsh environmental conditions including dust, rain, and extreme temperatures.
  • Multiple Cameras & Sensors:
    Supports front and rear high-resolution EO/IR cameras and other payloads for versatile mission profiles.
  • Remote Command & Control:
    Fully integrated with Anduril's Lattice OS, enabling remote command, asset coordination, autonomous tasking, and mission handoff between operators or systems.
  • Multi-Modal Flight Operation:
    Supports hover, loiter, and forward flight, making it suitable for urban, rural, and complex terrain reconnaissance.
  • Modular & Compact Airframe:
    Folding-arm quadcopter design with bottom-mounted motors allows for a compact stowed profile without compromising stability or payload capability.

Use Cases

  • Tactical overwatch during dismounted patrols or convoy movement
  • Route clearance and IED reconnaissance
  • Urban rooftop surveillance
  • Disaster response and search-and-rescue support
  • Remote inspection of hazardous environments

Technical Highlights

  • Airframe Layout: 3-segment design:
    • Bottom ¼: camera, gimbals, and video transmission
    • Centre ½: folding arms, batteries, ESCs
    • Top ¼: flight controller, FPV camera, and sensors

Proof of Concept Specifications

  • Propulsion: A2212/6T 2200KV brushless motors (bottom-mounted)
  • Avionics: ESP32-S3 flight controller with purpose-built firmware
  • Processing: Raspberry Pi companion computer for autonomy and vision tasks
  • Sensors:
    • MPU-6050 IMU
    • HMC5883L magnetometer
    • BMP388 barometer
    • NEO-6M GPS
  • Communication: SCP (Secure Control Protocol) over UART/USB/SPI
  • Payload: High-quality camera module with stabilised gimbal

Integration

Scout communicates with mission systems via the Secure Control Protocol (SCP). Integration with other autonomous assets can be provided via external systems (e.g., Lattice OS) using SCP bridges.

Developer setup (firmware)

This repo includes an ESP‑IDF firmware for ESP32‑S3 built with PlatformIO.

Prerequisites

  • Virtual Environment: This project uses a Python virtual environment for PlatformIO and dependencies.

    IMPORTANT: Always activate the virtual environment before working with the project:

    cd /Users/paulzanna/Github/Scout
    source .venv/bin/activate

    Note for new terminals: Remember to activate the virtual environment in every new terminal session before running any PlatformIO commands.

Installation

  • Install PlatformIO:
    • VS Code extension (recommended), or
    • macOS CLI: brew install platformio (or pipx install platformio)
  • Build the firmware: run the PlatformIO build for env freenove_esp32_s3_wroom.
  • Monitor serial at 115200 baud.

Board/profile defaults are in platformio.ini. Core tasks are created in src/main.c and foundational modules live under src/{control,services,cli} with public headers in include/.
