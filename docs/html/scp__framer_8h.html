Detailed Description

Public API for SCP framing (COBS) and CRC utilities.

Provides encode/decode helpers for SCP frames using:

  • Fixed header defined in scp_protocol.h
  • Optional auth tag (SCP_AUTH_TAG_SIZE)
  • CRC-16-CCITT trailer
  • COBS outer framing (zero-terminated)
