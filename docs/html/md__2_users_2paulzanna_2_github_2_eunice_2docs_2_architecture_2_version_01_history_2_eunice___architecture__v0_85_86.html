Eunice Architecture — v0.5.6

Title: Evidence Synthesis & Report Generation (Qualitative + Basic Meta‑analysis)
Status: Planned
Date: 11‑08‑2025 (Australia/Melbourne)
Depends on: v0.5.2 (retrieval/dedup/ranking/queue/PRISMA), v0.5.3 (auto‑screen, full‑text, parsing), v0.5.4 (extraction v2), v0.5.5 (critical appraisal)

1) Objective

Produce a fully automated literature synthesis and thesis‑style report with selectable rigour levels (Undergraduate / Masters / PhD). From extracted and appraised studies, the system must:

  • Group comparable studies into claim sets by outcome/measure/population and compute aggregate effects when feasible.
  • Generate a narrative synthesis that only states claims supported by citations ("no cite, no claim").
  • Emit artefacts: PRISMA, forest plots, evidence tables, bibliography, and a reproducibility bundle (configs, hashes, versions).

Out of scope: advanced publication bias modelling (funnel plots, trim‑and‑fill), meta‑regression, network meta‑analysis, UI.

2) High‑level Flow

Starting from records included by v0.5.3, extracted by v0.5.4, and appraised by v0.5.5:

1) Eligibility gating by appraisal tier and level profile.
2) Claim grouping: cluster extractions into comparable sets (same outcome, compatible measures/units/design).
3) Effect size calculation per study arm/contrast; unit harmonisation.
4) Aggregation across studies (fixed‑effect baseline; PhD level may also compute random‑effects DL).
5) Narrative synthesis conditioned on level profile.
6) Report assembly with figures/tables/bibliography and a zip repro bundle.

3) Scope (Detailed)

In scope

  • Compatibility rules and claim‑set formation.
  • Effect size calculators (continuous and binary) + variance.
  • Aggregation (fixed effect; optional DL random at PhD level) + heterogeneity metrics (Q, I²).
  • Narrative generator with strict citation policy and evidence tables.
  • Artefact generation (forest plots, CSV tables, .bib, markdown/PDF report) and repro bundle.

Out of scope

  • Meta‑regression and subgroup analysis beyond basic stratification by design/species.
  • Publication bias diagnostics beyond simple counts and optional Egger test (PhD: optional, not required).

5) Compatibility & Claim Grouping

A claim set is defined by a compatibility signature:

  • outcome (normalised label, e.g., spike_rate, neurite_length, viability).
  • measure (e.g., mean with SD; proportion; odds ratio).
  • population (species/cell_line/organ) and design (in vitro / in vivo / clinical).
  • timepoint window (e.g., DIV±2 days) and unit after normalisation.
  • contrast structure (2‑arm control vs treatment; multi‑arm reduced to independent contrasts if design allows).

Records failing compatibility become qualitative‑only evidence.

6) Effect Size Calculators

Implement deterministic calculators with input validation. Bail out to qualitative on invalid/missing inputs. Use unbiased or commonly accepted estimators, log‑scale for ratios, and delta methods for variances.

6.1 Continuous outcomes (two‑group)

  • Mean Difference (MD):
    ( \hat{\Delta} = \bar{x}_T - \bar{x}_C ),
    ( \mathrm{Var}(\hat{\Delta}) = s_T^2/n_T + s_C^2/n_C ).
  • Standardised Mean Difference (Hedges g):
    Pooled SD: ( s_p = \sqrt{\frac{(n_T-1)s_T^2 + (n_C-1)s_C^2}{n_T+n_C-2}} ).
    Cohen's d: ( d = (\bar{x}_T - \bar{x}_C)/s_p ).
    Small‑sample correction: ( J = 1 - \frac{3}{4(n_T+n_C)-9} ), ( g = J\,d ).
    Variance (approx.): ( \mathrm{Var}(g) = \frac{n_T+n_C}{n_T n_C} + \frac{g^2}{2(n_T+n_C-2)} ).
  • Choose MD when units are homogeneous; otherwise prefer g. Convert units pre‑calculation.

6.2 Binary outcomes (two‑group)

  • Log Risk Ratio (log RR): ( \hat{\theta} = \log\frac{a/(a+b)}{c/(c+d)} ),
    ( \mathrm{Var}(\hat{\theta}) = \frac{1}{a} - \frac{1}{a+b} + \frac{1}{c} - \frac{1}{c+d} ).
  • Log Odds Ratio (log OR): ( \hat{\theta} = \log\frac{ad}{bc} ),
    ( \mathrm{Var}(\hat{\theta}) = \frac{1}{a} + \frac{1}{b} + \frac{1}{c} + \frac{1}{d} ).
  • Apply continuity corrections when any cell is zero (configurable, default 0.5).

6.3 Confidence Intervals

For any effect (\hat{\theta}) with variance (\hat{V}):
CI: ( \hat{\theta} \pm z_{1-\alpha/2}\sqrt{\hat{V}} ) (default (\alpha=0.05)). Transform back from log scale as needed.
