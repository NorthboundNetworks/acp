Detailed Description

Real-time performance guardrails implementation.

Prevents arming and degrades features when CPU budget exceeded per Constitution requirements (≤70% total CPU, loop rates within ±5%).

Features:

  • CPU budget monitoring and enforcement
  • Loop rate tolerance checking
  • Automatic feature degradation before critical failure
  • Integration with safety manager for arming blocks
