Command Schemas (text contracts)

version

  • Inputs: none
  • Output (text):
    • firmware: SEMVER
    • api: SEMVER
    • build: GIT_SHA/DATE
  • Errors: none

status

  • Inputs: none
  • Output (text):
    • mode: CONFIG|FLIGHT
    • armed: true|false
    • safety: arming_allowed=BOOL cpu_budget_ok=BOOL
    • sensors: imu=OK|FAULT mag=OK|FAULT baro=OK|FAULT gps=NOFIX|2D|3D
    • battery: v=V i=A rem=PCT (if available)
    • link: quality=0-100
  • Errors: none

stats

  • Inputs: none
  • Output (text):
    • loops:
      • LOOP_NAME: target=HZ measured=HZ p95=MS
    • cpu: total=PCT
  • Errors: none

dump

  • Inputs: none
  • Output (text): stable, sorted key=value lines; header with checksum
    • header: checksum=HEX count=N
    • body: SECTION.KEY=VALUE
  • Constraints: max dump size ~8 KB; round‑trip importable; complete representation
  • Errors: none

get

  • Inputs: PARAM
  • Output (text): KEY=VALUE
  • Errors: 400 invalid param name; 404 unknown param

set

  • Inputs: PARAM VALUE [...]
  • Output (text): OK
  • Errors: 400 invalid value/schema; 404 unknown param; 409 read-only param

save

  • Inputs: none
  • Output (text): SAVED (and persisted)
  • Errors: 507 insufficient storage; 500 persistence error

reboot

  • Inputs: none
  • Output (text): REBOOTING
  • Errors: none

cal (calibration helpers)

  • Inputs: cal imu | cal mag
  • Output (text): step-by-step prompts and final RESULT=OK|FAIL with reason
  • Errors: 409 busy/armed; 500 calibration error

Configuration Import/Export Boundaries

  • Storage: ESP-IDF NVS partition (default 24–64 KB); keys under namespace cfg.*
  • Atomicity: write-then-commit per key; save batches critical keys with checksum
  • Size constraints: total config payload ≤ 6 KB; single value ≤ 512 B
  • Dump format: stable order, newline-delimited KEY=VALUE, ASCII; checksum over body
  • Compatibility: unknown keys ignored on import; invalid values rejected with clear errors
