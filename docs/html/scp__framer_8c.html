Detailed Description

SCP Protocol Frame Processing - COBS Encoding/Decoding and Message Validation.

Implements Consistent Overhead Byte Stuffing (COBS) framing, CRC-16-CCITT validation, and message parsing for the Secure Control Protocol (SCP) v1.1.

COBS Encode Data.
