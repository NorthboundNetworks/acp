Detailed Description

CRC16-CCITT implementation for ACP frame integrity.

This module implements CRC16-CCITT (ITU-T X.25) with polynomial 0x1021, initial value 0xFFFF, and no final XOR. This provides frame integrity checking for ACP protocol frames.
