Companion Input Mediation Integration Test

Purpose

Validate companion input mediation policy per FR-011 to ensure companion cannot bypass safety gates or timing constraints. All companion inputs must be subject to flight controller safety validation.

Test Environment

  • Hardware: ESP32-S3 development board
  • Companion: Simulated or actual companion computer
  • Connection: UART/serial link for companion communication
  • Test modes: Both CONFIG and FLIGHT modes
  • Prerequisites: Propellers removed, bench setup
