[0.1.0] - 2024-XX-XX - Initial MVP Release

Initial Release Features

  • Serial Communication: POSIX termios-based serial port interface
    • Support for standard baud rates (9600 to 230400)
    • Non-blocking I/O with configurable timeouts
    • Connection monitoring and reconnection handling
  • SCP Frame Decoding: Complete SCP v1.1 protocol implementation
    • State machine-based decoder with byte stuffing support
    • CRC16-CCITT validation for data integrity
    • Support for DATA, ACK, NAK, HEARTBEAT, and CONFIG frames
    • Frame statistics and error reporting
  • Display Modes: Flexible output formatting
    • ASCII table mode with configurable columns
    • Optional ncurses TUI support (when available)
    • Real-time updates with configurable refresh rates
    • Timestamp display with multiple formats
  • Data Logging: Structured telemetry persistence
    • CSV format with customizable column definitions
    • Buffered and immediate write modes
    • System logging with multiple severity levels
    • Automatic log rotation and file management
  • Configuration Management: Flexible runtime configuration
    • INI-style configuration files with sections
    • Command-line argument processing with getopt_long
    • Environment variable support
    • Runtime parameter validation
  • Build System: Robust development infrastructure
    • Makefile with release/debug configurations
    • Cross-platform support (Linux/macOS)
    • Static analysis integration (clang-tidy, cppcheck)
    • Memory sanitizers and debugging tools
    • Unity test framework integration
  • Core Features:
    • US1: Connect to serial device and display live telemetry
    • US2: Log telemetry data to CSV files
    • US3: Configuration file and command-line options

Technical Details

  • Language: C11 (ISO/IEC 9899:2011) with POSIX extensions
  • Dependencies: Standard C library, POSIX termios, optional ncurses
  • Platforms: Linux, macOS (Windows support planned)
  • License: MIT License
  • Build Requirements: GCC/Clang, Make, pkg-config
