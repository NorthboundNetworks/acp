Detailed Description

Minimal runtime GCS Gateway implementation aligned with current SCP protocol.

Implements the subset of GCS gateway APIs used by the CLI today:

  • Statistics getters/reset
  • Telemetry rate/enable controls and stats (T038)
  • Per-session telemetry configuration and stats (T039)
  • Priority queue stats stub (T040)
  • TLM_STATUS generation and send-to-sessions (no transport wired yet)

Notes:

  • Uses current scp_protocol.h header layout and scp_framer API
  • Frames are encoded without auth tag (flag not set). Higher-layer HMAC integration can be added later by generating SCP_AUTH_TAG and setting SCP_FLAG_AUTH_PRESENT in header->flags.
