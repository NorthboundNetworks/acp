Data Model

Date: 2025-10-26

Entities

CompanionInput

  • ts: int64 (ms since boot or UTC FC timebase)
  • source_identity: string (SYSID/COMPID or equivalent)
  • classification: enum {command, suggestion}
  • payload_summary: string (human-readable)
  • correlation_id: string (optional)
  • decision: enum {allowed, rejected}
  • reason_codes: string[] (e.g., ["disarmed", "geofence"])

SafetyEvaluation

  • ts: int64
  • gates_evaluated: string[] (e.g., ["armed", "mode", "geofence", "alt_limit", "failsafe"])
  • per_gate: map<string, enum {pass, fail}>
  • decision: enum {allowed, rejected}
  • reasons: string[]

BlackboxEvent

  • ts: int64
  • type: enum {arm, disarm, mode_change, failsafe, sensor_summary, operator_cmd, companion_input, gimbal_update, system}
  • severity: enum {info, warn, error} (optional)
  • attrs: object (free-form key/values, see NDJSON schema)
  • schema_version: string (e.g., "1.0")

MissionLog

  • mission_id: string (UTC start timestamp or monotonic id)
  • start_ts: int64
  • end_ts: int64 (set on landing)
  • duration_s: int32
  • size_bytes: int32
  • integrity: enum {ok, partial, corrupted}
  • events: [BlackboxEvent] (not necessarily materialized in memory)

GimbalState

  • ts: int64
  • mode: enum {fixed, follow_heading, search_sweep}
  • target_pitch_deg: float
  • target_yaw_deg: float
  • actual_pitch_deg: float
  • actual_yaw_deg: float
  • slew_limit_deg_s: float
  • sweep_min_yaw_deg: float
  • sweep_max_yaw_deg: float
  • sweep_rate_deg_s: float
  • fault_flags: string[] (e.g., ["overcurrent", "limit"])

Relationships

  • MissionLog contains BlackboxEvents
  • BlackboxEvent may reference CompanionInput, SafetyEvaluation, or GimbalState in attrs

Validation Rules

  • CompanionInput.decision must match SafetyEvaluation.decision
  • Gimbal sweep bounds must satisfy min < max and |bounds| ≤ 270° by default
  • Sensor summary cadence default = 2s; configurable via CLI (bounded)
  • Export requires HMAC metadata present if integrity == ok

State Transitions

  • MissionLog: created → recording → completed (on disarm/landing) → exported
  • GimbalState: fixed ↔ follow_heading ↔ search_sweep (explicit CLI/MAVLink commands)
