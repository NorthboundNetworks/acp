Detailed Description

MAVLink arm/disarm command handling interface.

Provides MAVLink COMMAND_LONG processing for MAV_CMD_COMPONENT_ARM_DISARM with full safety gate integration and proper COMMAND_ACK responses.

mavlink_command_result_t cmd_arm_disarm_handle_command_long ( const mavlink_command_long_t * cmd)

Process MAVLink COMMAND_LONG message.

Handles incoming MAVLink commands, with special processing for:

  • MAV_CMD_COMPONENT_ARM_DISARM (400): Arm/disarm autopilot
  • MAV_CMD_NAV_TAKEOFF (22): Takeoff command (placeholder)
  • MAV_CMD_NAV_LAND (21): Land command (placeholder)

cmdPointer to COMMAND_LONG message structure
Returns
Command processing result for COMMAND_ACK response
