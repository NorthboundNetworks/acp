Detailed Description

ACP cryptographic functions - SHA-256 and HMAC-SHA256.

Provides portable implementations of SHA-256 hash and HMAC-SHA256 message authentication code functions for the ACP protocol.

Features:

  • Pure C99 implementation, no external dependencies
  • Constant-time operations for security-critical functions
  • Self-test vectors for validation
  • Optimized for embedded and cross-platform use
