Detailed Description

ACP cryptographic functions - SHA-256 and HMAC-SHA256.

This module provides portable HMAC-SHA256 and SHA-256 implementations for the ACP protocol authentication system.

All functions are implemented in pure C99 without external dependencies, implementing RFC 2104 (HMAC) and FIPS 180-4 (SHA-256).
